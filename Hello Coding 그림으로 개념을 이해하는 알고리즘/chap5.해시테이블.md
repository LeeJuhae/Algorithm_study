## Chap5. 해시 테이블

## 02 해시 함수

해시 함수는 문자열을 받아서 숫자를 반환하는 함수이다. 

해시 함수와 배열을 합치면 **해시 테이블**이다. 해시 테이블은 배열을 사용하기 때문에 속도가 빠르다.

## 05 성능

|      | 해시 테이블(평균적인 경우) | 해시 테이블(최악의 경우) | 배열 | 연결 리스트 |
| :--: | :------------------------: | :----------------------: | :--: | :---------: |
| 탐색 |            O(1)            |           O(n)           | O(1) |    O(n)     |
| 삽입 |            O(1)            |           O(n)           | O(n) |    O(1)     |
| 삭제 |            O(1)            |           O(n)           | O(n) |    O(1)     |
