### Chap11. 더 공부해야 할 것

### 01 트리

이진 탐색은 정렬된 배열에 대해서만 사용 가능하므로 원소 삽입시 재 정렬 해야한다. 그래서 이진 탐색 트리라는 자료구조가 만들어지게 되었다. 

|      |   배열   | 이진 탐색 트리 |
| ---- | :------: | :------------: |
| 탐색 | O(log n) |    O(log n)    |
| 삽입 |   O(n)   |    O(log n)    |
| 삭제 |   O(n)   |    O(log n)    |

- B-트리(B-trees)
- 레드-블랙 트리(red-black trees)
- 힙(heaps)
- 스플레이 트리(splay trees)

### 03 퓨리에 변환

퓨리에 변환은 매우 많은 응용 분야를 가지는 희귀한 알고리즘이다. 이것을 사용하면 하나의 노래를 여러 개의 개별적인 주파수들로 분리 가능하다.
